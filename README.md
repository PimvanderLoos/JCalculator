## Supported operators
+, -, *, /, %, ^, min(x,y), max(x,y), abs(x), sqrt(x).

## Supported characters
'(', ')', ','

## Examples
- 1+1 = 2
- sqrt(144) = 12
- max(10, sqrt(16)^4/100*4) = 10.4

With x = 2:
- x\*x\*x  = 8
